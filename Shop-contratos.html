Función publicar:

El cliente debe enviar al portafolio web 24 imágenes web para el archivo web, luego de aceptar las condiciones para ser publicadas en la plataforma web shop. Al enviar las imágenes web en formato .jpeg identificar cual es el país, zona: interior y costera, la ciudad, el estado y la imagen de portada que se va a colocar en el área de la portada del álbum web . Al terminar el hospedaje por tiempo determinado de las imágenes web participantes pasan a ser parte de la plataforma web shop para uso en el cuadro de tiempo permanente de imágenes web.
